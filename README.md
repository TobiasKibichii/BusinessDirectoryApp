# SUMMARY
A cross-platform mobile application designed to help users discover, explore, and interact with local businesses across multiple categories such as coffee shops, grocery stores, gyms, and restaurants. The app focuses on intuitive navigation, personalized discovery, and real-world usability.

# TECH STACK.
<ol>
<li>React Native (Expo).</li>
<li<Clerk Authentication.</li>
<li>SQL Database (XAMPP). </li>
<li>RESTful APIs.</li>
</ol>

# KEY FUNCTIONALITIES
<ol>
<li> Categorized business listings for easy discovery</li>
<li>Popular businesses section based on user engagement.</li>
<li>Favorite system allowing users to save preferred businesses.</li>
<li>Search and explore functionality for quick access to businesses.</li>
<li>Detailed business profiles showing services, products, and offerings.</li>
<li>Secure user authentication and session management using Clerk.</li>
</ol>
